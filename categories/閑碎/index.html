妄思不可断

在稍熟悉的人看來，我可能就是個總愛成天胡思亂想、胡說八道，還老不動手的不著調傢伙。不可否認，想法多到來不及逐壹實現，確實是本人的壹大特點。這壹特點在帶給我痛處和遺憾的同時，也促使我不斷努力尋找行動超人做夥伴。然而不管怎樣，想法卻不能丟，這是天性使然。

在某壹段時間裏，我可能會處於相對“封閉”的狀態，甚至不得不讓自己陷入“孤陋寡聞”的境地。然而恰好是在這種“封閉”狀態下產生的某些妄思，卻總能與後來的了解到的某些產品或事物契合。諸多的“巧合”，讓我深切感受到錯失良機後搥胸頓足之痛，其中壹些案例，現在還記憶猶新。

Hello Github

壹直想搭建壹個比較私人的博客，讓自己天馬行空的想法和日常的絮絮叨叨都能有地方安放。雖說用Wordpress 搭網站已經輕車熟路，只用花很少的時間就能做好，不過考慮到後續要花不少時間來張羅服務器相關的事，所以始終沒動。正好知道越來越多的朋友在Github 上搭建自己的博客，作為久不寫代碼、更不懂Ruby 的偽Geek，這種方式仍然是我的優選。盡管花了壹些時間來摸索，但最終還是把這個博客搭起來了，確實很值得、很有成就感的說。

感謝Github及Octopress提供的各種資源和服務！
